Chatbot API

Welcome to the chatbot API documentation. Here you will find all the information you need to interact with our chatbot using HTTP requests.

Getting started

To use the chatbot API, you need to send HTTP requests to the following endpoint:

POST https://example.com/api/chatbot

Authentication

To access the chatbot API, you need to include an API key in your HTTP requests. You can obtain an API key by contacting our support team.

Sending messages

To send a message to the chatbot, send a JSON payload in the following format:

{
    "message": "Hello, chatbot!",
    "user_id": "1234",
    "api_key": "your_api_key"
}

The API will return a JSON response in the following format:

{
    "response": "Hi there, how can I assist you?",
    "bot_id": "5678",
    "timestamp": "2023-03-10T16:32:12Z"
}

Example code

Here is an example code snippet in Python that sends a message to the chatbot API:

import requests

url = "https://example.com/api/chatbot"

payload = {
    "message": "Hello, chatbot!",
    "user_id": "1234",
    "api_key": "your_api_key"
}

response = requests.post(url, json=payload)

print(response.json())
